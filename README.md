# 🌤️ Weather App - Previsão do Tempo

Este é um projeto desenvolvido para a disciplina de **Desenvolvimento Android**. O aplicativo permite que os usuários busquem e visualizem a previsão do tempo em tempo real para diferentes cidades.

## 🚀 Funcionalidades

* **Busca por Cidade:** Pesquisa de condições climáticas de qualquer cidade ao redor do mundo.
* **Dados em Tempo Real:** Exibição de temperatura atual, umidade, velocidade do vento e condição climática (ensolarado, nublado, chuva, etc.).
* **Favoritos:** [Opcional: se o seu app fizer isso] Possibilidade de salvar cidades para acesso rápido.

## 🛠️ Tecnologias Utilizadas

* **IDE:** Android Studio
* **Linguagem:** [Kotlin]

## 🏗️ Como Executar o Projeto

1.  Faça o clone deste repositório:
    ```bash
    git clone [https://github.com/seu-usuario/nome-do-repositorio.git](https://github.com/seu-usuario/nome-do-repositorio.git)
    ```
2.  Abra o projeto no **Android Studio**.
3.  Aguarde a sincronização do **Gradle**.
4.  (Importante) Pesquise chave API para conectar o mapa e os nomes das cidades
5.  Execute o app em um emulador ou dispositivo físico.
